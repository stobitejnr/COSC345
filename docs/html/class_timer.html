Detailed Description

The Timer class provides a countdown timer with start, pause, resume, reset, and time adjustment functionalities.

Default constructor prompts the user to set a timer with a specific duration.

The user can specify the duration in hours, minutes, and seconds. The timer is initialized and started based on the user input.
